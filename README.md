## MySQL Admin Service

Русскоязычный сервис администрирования баз данных MySQL с бэкендом на Node.js/Express/Sequelize и фронтендом на Vue 3/Vite/Tailwind.

### Возможности

- Первичная настройка и создание главного администратора
- JWT-аутентификация и разграничение прав (главный и обычные администраторы)
- Управление подключениями к серверам MySQL
- Создание/удаление баз, таблиц, CRUD по данным, выполнение запросов
- Создание резервных копий SQL
- Журналирование действий и метрики использования, аналитический дашборд
- Современный интерфейс в тёмной теме

### Запуск

```bash
# сервер
cd server
cp .env.example .env
npm install
npm run dev

# клиент
cd ../client
cp .env.example .env
npm install
npm run dev
```

В продуктиве запустите `npm start` в `server` (сборка не требуется) и `npm run build` в `client`, выдав статику через любой веб-сервер.
